ShotIQ — Run From Scratch

1. Clone the repository
   git clone https://github.com/Nalan1806/basketball-shot-predictor.git
   cd basketball-shot-predictor

2. Open the project folder
   In VS Code or File Explorer, open the cloned folder.

3. Create a virtual environment
   python -m venv venv

4. Activate the virtual environment on Windows PowerShell
   .\venv\Scripts\Activate.ps1

   If PowerShell blocks activation, run:
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
   .\venv\Scripts\Activate.ps1

5. Install dependencies
   pip install --upgrade pip
   pip install -r requirements.txt

   If ultralytics does not install automatically, install it separately:
   pip install ultralytics

6. Verify the code starts
   python main.py

7. First-time use
   - Allow the camera permission if Windows prompts for it.
   - Press C to calibrate the hoop if needed.
   - Drag the hoop box, then press ENTER to confirm.
   - Use R to reset tracking if needed.

8. Useful controls while running
   - ESC or Q: quit
   - C: calibrate hoop
   - R: reset tracker and trajectory
   - D: toggle debug overlay
   - S: save screenshot
   - SPACE: pause/resume

9. If something fails
   - Confirm the webcam is not being used by another app.
   - Check that the virtual environment is activated.
   - Reinstall dependencies with pip install -r requirements.txt
   - Make sure ultralytics is installed and the YOLO model can download.

10. Recommended launch check
   python -m py_compile main.py utils.py detector.py tracker.py ekf_tracker.py
   python main.py
